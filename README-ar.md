# ❤️ تطبيق التنبؤ بأمراض القلب Flask 💊

نظام تنبؤ طبي احترافي ومتعدد اللغات مبني باستخدام Flask والتعلم الآلي. تنبأ بخطر أمراض القلب باستخدام البيانات السريرية من خلال واجهة سهلة الاستخدام وبديهية.

## 🌍 عرض توضيحي مباشر

🔗 **[heart-disease-flask.onrender.com](https://heart-disease-flask-2ue3.onrender.com)**

## ✨ الميزات

### 🎯 الوظائف الأساسية
- **نموذج ML**: مصنف RandomForest للتنبؤ بأمراض القلب
- **متعدد اللغات**: الإنجليزية، الألمانية، العربية (دعم RTL)
- **إدخال بيانات طبية**: 13 معاملًا سريريًا
- **تقييم المخاطر**: 3 مستويات تقييم مخاطر مع ملاحظات بصرية
- **واجهة احترافية**: تصميم Teal حديث مع تخطيط متجاوب

### 📊 ميزات متقدمة
- **صناديق معلومات تفاعلية**: نظام معلومات ثلاثي المستويات (ما هو/ما تختار/المصدر)
- **شارات مصدر البيانات**: تشير إلى مصدر البيانات (تحليل دم، ECG، اختبار الإجهاد، متخصص)
- **تنبيهات طبية**: شارات ⚠️ للحقول التي تتطلب تقارير متخصصة
- **شريط التقدم**: تمثيل بصري لنسبة المخاطر
- **النتائج ملونة حسب المستوى**:
  - 🟢 أخضر (<40%) = خطر منخفض
  - 🟡 أصفر (40-70%) = خطر متوسط
  - 🔴 أحمر (>70%) = خطر مرتفع
- **التحقق من صحة النموذج**: التحقق من الحقول في الوقت الفعلي مع ملاحظات بصرية
- **زر إعادة التعيين**: مسح سريع للنموذج
- **عرض النتائج المحترف**: قسم إخلاء المسؤولية والتوصيات

### 🎨 التصميم
- مخطط ألوان Teal/Cyan حديث (#06b6d4)
- خلفيات متدرجة وظلال
- أيقونات طبية (❤️ 💊)
- رسوم متحركة وتحولات سلسة
- تصميم متجاوب للجوال
- دعم RTL للعربية

## 📋 الحقول الطبية

1. **العمر** - عمر المريض بالسنوات (29-77)
2. **الجنس** - ذكر/أنثى
3. **نوع ألم الصدر** - نموذجي/غير نموذجي/غير ذبحي/بدون أعراض
4. **ضغط الدم أثناء الراحة** - ضغط الدم بوحدة mm Hg
5. **الكوليسترول** - الكوليسترول في الدم بوحدة mg/dl
6. **سكر الدم الصائم** - مؤشر >120 mg/dl
7. **تخطيط القلب أثناء الراحة** - طبيعي/اضطراب ST-T/تضخم البطين الأيسر
8. **أقصى معدل لدقات القلب** - أعلى معدل أثناء التمرين
9. **ذبحة صدرية مع المجهود** - نعم/لا
10. **الانخفاض ST** - الانخفاض المستحثّ بالتمرين (0-6.2)
11. **ميل ST** - صاعد/مسطح/هابط
12. **عدد الأوعية الرئيسية** - العد من التصوير (0-4)
13. **الثال** - عادي/عيب ثابت/عيب قابل للعكس

## 🚀 البدء السريع

### المتطلبات الأساسية
- Python 3.8+
- pip أو conda

### الإعداد المحلي

```bash
# استنساخ المستودع
git clone https://github.com/Ahmed-Dadoue/heart-disease-flask.git
cd heart-disease-flask

# إنشاء بيئة افتراضية
python -m venv .venv
source .venv/bin/activate  # على Windows: .venv\Scripts\activate

# تثبيت المكتبات
pip install -r requirements.txt

# تشغيل التطبيق
python src/app_flask.py
```

زيارة: **http://localhost:5000** 🌐
